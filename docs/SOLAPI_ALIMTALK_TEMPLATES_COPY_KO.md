# WizCoCo — Solapi 알림톡 템플릿 등록용 (복사·붙여넣기)

Solapi 콘솔 → **카카오톡** → **알림톡 템플릿** → **템플릿 등록**  
아래 3건을 **각각** 등록하세요. 변수명은 반드시 그대로 사용합니다.

---

## 검수 반려 대응 (2차 수정 — 2026-07)

카카오 심사는 **「수신자가 먼저 한 행동」** 이 문장 **첫 부분**에 드러나야 합니다.  
`배정받으신`·`발급받으신` 만으로는 부족할 수 있습니다.

| 반드시 넣을 표현 (카카오 예시) |
|-------------------------------|
| **접수하신**, **신청하신**, **요청하신**, **예약하신** |

| Solapi 등록 시 추가 입력 |
|-------------------------|
| **발송 시점/용도** 란 → 아래 각 템플릿의 「발송 시점」 복사 |
| **심사 요청 메모** (있으면) → 「심사 메모」 문구 복사 |
| **변수 예시값** → 아래 「변수 예시」 표 참고 (미입력 시 반려) |

**반려된 템플릿** → 내용 수정 후 **재심사 요청** 또는 **새 템플릿** 등록.  
승인 후 **새 templateId** → `backend/.env.solapi` → `npm run secrets:solapi`

## 템플릿 1 — 검사 미완료 리마인더

- **템플릿 이름:** WizCoCo 검사 미완료 안내
- **카테고리:** 기타 / 안내
- **변수:** `#{name}`, `#{title}`, `#{pending}`, `#{link}`
- **발송 시점:** 내담자가 WizCoCo 심리검사 참여를 접수한 뒤, 검사가 미완료일 때 상담사 시스템이 자동 발송
- **심사 메모:** 수신자는 전문상담사 연계로 검사 참여를 **접수**한 내담자이며, 본 메시지는 미완료 검사 이어하기 안내입니다. 광고·마케팅 목적이 아닙니다.

**변수 예시**

| 변수 | 예시값 |
|------|--------|
| `#{name}` | 홍길동 |
| `#{title}` | 우울 척도 검사 |
| `#{pending}` | 미완료 1건 |
| `#{link}` | https://wizcoco.com/portal/login/ |

```
안녕하세요 #{name}님,
고객님께서 접수하신 WizCoCo 심리검사가 아직 완료되지 않았습니다.

검사명: #{title}
미완료: #{pending}

이어서 진행: #{link}
```

→ `SOLAPI_KAKAO_TEMPLATE_TEST_REMINDER`

## 템플릿 2 — 치료·과제 할당

- **템플릿 이름:** WizCoCo 치료·과제 안내
- **변수:** `#{name}`, `#{title}`, `#{link}`
- **발송 시점:** WizCoCo 상담·검사 서비스를 이용 접수한 내담자에게, 담당 상담사가 치료·과제를 등록했을 때 발송
- **심사 메모:** 수신자는 서비스 **이용 접수** 내담자입니다. 상담사가 등록한 치료·과제 **내용 안내**이며 광고가 아닙니다.

**변수 예시**

| 변수 | 예시값 |
|------|--------|
| `#{name}` | 홍길동 |
| `#{title}` | 호흡 이완 과제 1회 |
| `#{link}` | https://wizcoco.com/portal/?tab=care |

```
안녕하세요 #{name}님,
WizCoCo 상담·검사 서비스 이용을 접수하신 고객님께 안내드립니다.
담당 전문상담사가 등록한 치료·과제입니다.

#{title}

바로 보기: #{link}
```

→ `SOLAPI_KAKAO_TEMPLATE_CARE`

## 템플릿 3 — 포털 자격증명 (나의코드·PIN)

- **템플릿 이름:** WizCoCo 검사 접속 안내
- **변수:** `#{name}`, `#{joincode}`, `#{mycode}`, `#{pin}`, `#{link}`
- **발송 시점:** 내담자가 WizCoCo 심리검사 참여를 접수한 뒤, 담당 상담사가 포털 접속 정보를 발송할 때
- **심사 메모:** 수신자가 **접수한** 검사 참여 건에 대한 나의코드·PIN **접속 정보** 전달입니다. 광고가 아닙니다.

**변수 예시**

| 변수 | 예시값 |
|------|--------|
| `#{name}` | 홍길동 |
| `#{joincode}` | ABC123 |
| `#{mycode}` | XYZ789 |
| `#{pin}` | 1234 |
| `#{link}` | https://wizcoco.com/portal/login/ |

```
안녕하세요 #{name}님,
고객님께서 접수하신 WizCoCo 심리검사 참여에 대한 접속 정보입니다.

검사코드: #{joincode}
나의코드: #{mycode}
비밀번호: #{pin}

바로 시작: #{link}
```

→ `SOLAPI_KAKAO_TEMPLATE_PORTAL_CREDENTIALS`

## 등록 체크리스트

1. **첫 문장**에 `접수하신` 또는 `신청하신` 포함 여부
2. 변수 `#{...}` 이름·개수가 위와 **완전히 동일**
3. 각 변수에 **예시 텍스트** 입력 (Solapi UI)
4. 발송 시점·심사 메모 입력
5. 광고·이벤트·채널추가 유도 문구 **없음**
6. 3종 승인 후 `npm run secrets:solapi`
